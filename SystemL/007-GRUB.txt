现在版本系列为2系列，比老的GRUB legacy版本0.97有很大的改进。

Features:
* Basic scripting support
* GUI (better bootsplash support, custom colors, custom themes, ...)
* Memory management
* Cleaner design
* Better portability
* Internationalization
* Rescue mode
